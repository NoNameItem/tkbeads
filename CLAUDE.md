## Project Overview

tkbeads is a Python project requiring Python 3.14+. The project uses a virtual environment located in `.venv/`.

## Development Setup

```bash
# Activate virtual environment
source .venv/bin/activate

# Run the application
python main.py
```

## Build System

This project uses `pyproject.toml` for configuration. Dependencies can be managed using pip or uv.
