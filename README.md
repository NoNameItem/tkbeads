# tkbeads

TUI-клиент для [Beads](https://github.com/steveyegge/beads) — распределённого трекера задач для AI-агентов.

## Возможности

- Kanban-доска с 4 колонками: Blocked, Ready, In Progress, Closed
- Древовидное отображение задач с parent-child связями
- Смена статуса задач
- Поиск по названию и переход по ID
- Автообновление при изменении файла извне

## Установка

```bash
# Требуется Python 3.14+
pip install tkbeads
```

## Использование

Запустите в директории с проектом Beads (где есть `.beads/`):

```bash
tkbeads
```

## Навигация

| Клавиша | Действие |
|---------|----------|
| `Tab` / `Shift+Tab` | Переключение между колонками |
| `↑` / `↓` | Выбор задачи |
| `Enter` | Открыть детали задачи |
| `Space` | Свернуть/развернуть узел |
| `/` | Поиск по названию |
| `g` | Переход по ID |
| `q` | Выход |
